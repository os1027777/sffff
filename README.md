### git 명령어
-init : 저장소 생성
- add : 작업한 파일 or 폴더 추가
-commit : 작업한 파일 or 폴더 저장
- push : 최종 github에 업로드
-clone : 다운로드
-pull : 변경된 내용 다운로드
